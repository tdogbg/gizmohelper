# Gizmo Helper

Welcome to the Gizmo Helper! This is a simple double-click pop-up application designed for easy use without requiring any technical knowledge.

## How to Get Started
1. **Download the Repository**  
   - Click the green "Code" button on the repository page to download the zip file.
   - Extract the contents of the zip file to a folder on your computer.

2. **Add Your Groq API Key**  
   - Locate the configuration file in the extracted folder where you need to input your Groq API key. Follow the instructions in the file to properly add your key.

3. **Run the Application**  
   - Simply double-click the application icon to run it. No need to open a terminal or perform any installations!

## How to Use the Pop-Up Window
- Once the app is running, a pop-up window will appear. You can use this pop-up to interact with the functionalities offered by the Gizmo Helper.
- Follow the on-screen instructions for using the features of the app.

## Troubleshooting
- If the application does not start:  
   Ensure that you have followed the steps to download and configure the app correctly.  
   If you still encounter issues, double-check that your Groq API key is entered correctly in the configuration file.

- For any other issues:  
   Please refer to the FAQ section or check the GitHub Issues page for support.
